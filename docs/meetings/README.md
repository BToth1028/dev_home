# Meeting Notes

Meeting notes, brainstorming sessions, decision logs.

## Structure

Organize by month: `YYYY-MM/YYYY-MM-DD_meeting-topic.md`

```
meetings/
├── 2025-10/
│   ├── 2025-10-15_api-design-review.md
│   └── 2025-10-26_architecture-discussion.md
└── 2025-11/
    └── 2025-11-05_sprint-planning.md
```

## Meeting Note Template

```markdown
# [Meeting Topic]

**Date:** 2025-10-26
**Attendees:** Names or "Solo"
**Type:** Planning / Review / Brainstorm / Decision

## Agenda
1. Topic 1
2. Topic 2

## Discussion

### Topic 1
[Notes]

### Topic 2
[Notes]

## Decisions Made
- [ ] Decision 1 (Action: Who, When)
- [ ] Decision 2 (Action: Who, When)

## Action Items
- [ ] Task 1 - @person - Due: YYYY-MM-DD
- [ ] Task 2 - @person - Due: YYYY-MM-DD

## Follow-up
- Next meeting: YYYY-MM-DD
- Related docs: [links]
```

## Maintenance

Archive old months periodically (move to `archive/` subfolder if needed).
